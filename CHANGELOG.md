## 0.1.0

### Features

- **Tauri v2 Desktop App** - Cross-platform desktop application (macOS, Windows, Linux)
- **React 19 + TypeScript** - Modern frontend stack with Vite 7
- **Theme System** - Light/dark mode with system preference detection
- **Global Shortcuts** - Configurable keyboard shortcuts (Cmd/Ctrl + ,)
- **Auto Start** - Optional launch on system startup
- **System Tray** - Minimize to tray with quick access menu
- **Settings Page** - User preferences management
- **Warm Academic Design** - Beautiful UI with Tailwind CSS + shadcn/ui
